## Installation

Install Bookshelf-Apps with npm

1. Clone the repo
   ```sh
   https://github.com/febrian2108/bookshelf-apps.git
   ```
2. Go to project
   ```sh
   cd bookshelf-apps
   ```
3. open project
   ```sh
   code .
   ```
4. run project
   ```sh
   http://localhost:5500/
   ```
